# Express server template

Create a server using:

- Express
- Typescript
- Routing Controllers
- Typeorm

---

## Requires degit

```bash
npm install -g degit
```

## Download template

```bash
degit Neo-Ciber94/template-express-routing-typeorm
```

## Install all the required node modules

```bash
npm install
```

## Run

```bash
yarn dev
```

Or

```bash
npm run dev
```
